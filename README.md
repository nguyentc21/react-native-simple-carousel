# react-native-simple-carousel

A simple carousel for React native app

## Installation

```sh
npm install react-native-simple-carousel
```

## Usage

```js
import { SimpleCarouselView } from "react-native-simple-carousel";

// ...

<SimpleCarouselView color="tomato" />
```
